# Doppelganger V1.1 — Frozen

Status: **local-first continuity kit + validated Markdown/Notion-like
import**.

Frozen on: **2026-06-20**.

## Proven boundary

V1.1 proves that a user can:

- create and revoke local continuity cards
- validate Markdown or Notion-like material before it becomes a card
- build a scoped context pack
- create and export a non-authoritative handoff
- inspect receipts and exact dry-run projections
- prepare a Quark-bound envelope without making a network call

The freeze protects these invariants:

- authored material is not a continuity object until Doppelganger validates it
- handoff is context, never command or autonomy
- fossil trace is structure, never narrative memory
- receipt records one past action, never permanent consent
- local revocation does not overclaim remote recall
- no network deposit exists in V1.1

## Freeze rule

V1.1 accepts only:

- security corrections
- portability and compatibility corrections
- data-loss or doctrine-enforcement corrections
- documentation corrections that do not widen capability

New transport behavior belongs to V2.
